# Rent-An-Anime-Sito-Web-
Sito web creato con html, css, php e javascript e hostato localmente con xampp per sviluppo e test.

E' un sito e-commerce, dove si possono noleggiare anime. Il sito è stato sviluppato in modo da potersi registrare creando un account, poter scegliere tra varie categorie per selezionare il tipo di anime che si vuole noleggiare, leggere le varie informazioni, guardare il trailer ufficiale, poter tenere nel carrello i vari anime, poterli acquistare scegliendo il metodo di pagamento e poter tenere traccia della cronologia degli acquisti.

--------------------------------------------------------------------------------

Per svilupparlo e testarlo ho usato come host "xampp".

Per provarlo, bisogna creare un database nel cmd (prompt dei comandi) con il nome scritto sul file "database" nella cartella "database" (in questo caso "Rent_An_Anime"), usarlo e creare le varie tabelle (come nel file "database").
Dopo aver creato il database e le cartelle, bisogna riempire le tabelle (i dati sono nel file "insert" della cartella "database").

Per visualizzare il sito web:
1. Bisogna posizionare la cartella del sito "Rent_An_Anime" nella cartella "htdocs" (percorso: "C:\xampp\htdocs")
2. Avviare Xampp e Apache
3. Incollare nella barra di ricerca: http://localhost/Rent_An_Anime
